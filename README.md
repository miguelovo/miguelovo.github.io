## About this repository

This repository powers **[miguelovo.github.io](https://miguelovo.github.io/)**, my professional and academic website. It brings together my research profile, publications, technical reports, conference work and software related to electric-power-system operations and planning.

## Research focus

- **Reserve sufficiency and operational risk** — probabilistic reserve sizing, intraday reliability and uncertainty-aware operations.
- **Coordinated system planning** — reliability-, resilience- and flexibility-informed expansion planning.
- **Flexibility and emerging large loads** — data centres, storage, flexible demand and grid-aware operation.
- **Decision analytics under uncertainty** — probabilistic forecasting, stochastic optimisation and risk-aware decision support.
- **Electricity markets and system operations** — unit commitment, production-cost modelling and market applications.

## Using this repository

The site is built with **GitHub Pages and Jekyll**.

```bash
# Clone the repository
git clone https://github.com/miguelovo/miguelovo.github.io.git
cd miguelovo.github.io

# Install dependencies and run locally
bundle install
bundle exec jekyll serve
```

Then open `http://localhost:4000` in a browser.

Most content is maintained as Markdown pages. Site-wide settings are controlled through `_config.yml`; reusable layout elements are under `_includes`; images and supporting files are stored under `Files` and `assets`.
